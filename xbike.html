XBike Tracker

XBike Tracker is a dashboard for visualizing endurance sports data such as exercise bike data.

Goal

The main requirement of this web application was being able to convert an endurance sports dataset (in .csv format) into something that is more visually appealing and motivates the user to exercise more frequently and to improve their personal records. A mobile version of the dashboard was also required.

Result

XBike Tracker was developed using D3.js, a Javascript library for visualizing data, and displays various metrics and graphs. The graphs include a bar graph displaying the covered distance on a monthly basis, a scatter plot showing the average speed per exercise session as well as a pie chart with the distribution of distances per week day. The metrics and graphs about distance motivate the user to sport more frequently and to extend the exercise duration, whereas the metrics and graphs about speed motivate the user to beat previous speed records.

Techniques

  • D3
  • HTML5
  • CSS3
  • JavaScript
